Hexo博客的搭建

发表于 2016-03-10   |   分类于 Blog   |  

什么是Hexo

Hexo是台湾一位叫tommy351的大牛开发的开源博客框架。就两字儿清爽。

为什么选择Hexo

最重要的是可以免费托管在GitHub上。之前考虑过弄台vps在上面用WordPress来做博客。但是像我这样只需在上面日常写写，记录下生活，一个静态的足以。而且Hexo比Wordpress轻巧许多。

以下是安装方法（本人是在Windows环境下，Mac用户可参考）

下载软件

  • GitHub For Windwos下载地址 https://desktop.github.com
  • Node.js 下载地址 https://nodejs.org/en/

安装软件

  • 两个软件下载以后全部下一步直到结束。

注册GitHub用户，添加仓库

  • 去官方注册用户名
  • 注册好以后登录，建立一个仓库
  • 设置Repostitory name，这个名字就是你后续通过浏览器访问的地址，记得名字后面一定要加 .github.io，否则可能造成访问不到

4:使用Git For Windows登录Git

  • 打开点击右上角小齿轮

  • 填入你的名字与邮箱

  • 输入申请的邮箱与密码登录

配置SSH KEY

  • 打开Git For Windwos里面的Git shell 输入ssh -T git@github.com 看到以下的页面就成功了！
